Technical documentation of Parcel Ascent Tracing System (PATS) - the numerical model for convective parcel ascent simulation in three-dimensions.

This documentation provides a description of functions and structures used in the model. Its main purpose is to make it easier to maintain and contribute to the project codebase. However, it can be also useful for users who want to understand the model in more detail.
